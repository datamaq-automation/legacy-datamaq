# Indice de Documentación Técnica - Migración Estética

Este directorio contiene la documentación técnica necesaria para preparar la migración estética del sitio implementado en Vue hacia la plataforma WordPress existente. El objetivo es asegurar la fidelidad visual, funcional y de composición entre ambas implementaciones.

## Propósito de la documentación
Cada documento de vista (`docs/*.md`) detalla la estructura, jerarquía, composición visual y comportamiento de una página completa, identificando brechas críticas frente al sitio activo en WordPress (`https://datamaq.com.ar`).
